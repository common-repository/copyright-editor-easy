== Description ==
Easily edit the footer copyright with the Copyright Editor plugin. You can remove the footer copyright or edit it to your liking. Free forever plugin for wordpress website!!

= What functions are available =
1. Remove footer copyrights
2. Edit copyright content in the footer

== Installation ==
1. Upload folder \'copyright-editor-easy-pht\' vào wp-content/plugins
2. Activate the plugin in the area Plugins -> Installed Plugins
3. Go to the Copyright Editor on the left side of the Menu to install.
